# NUTRICLASS_FOOD_CLASSIFICATION
-> DESCRIPTION: "EVALUATION OF FOOD NUTRITIONAL DATA USING VARIOUS MACHINE LEARNING MODELS IN PYTHON PANDAS."

-> FEATURES: 1. CSV file containing nutritional data loaded, cleaned and EDA done before building pipeline models. 2. Feature Engineering done and features are selected for prediction of Target variable. 3. Pipeline for various models like Logistic Regression, Decision Tree, Random Forest, K-Nearest Neighbours, Support Vector Machine, XG Boost and Gradient Boosting classifier are evaluated and classification report is generated. 

-> Tools Used: 1. Python, Pandas, Matplotlib and Seaborn (for visualization) and Scikit-Learn (for machime learning models)

-> Install tools mentioned in requirements.txt file.. using command pip install
